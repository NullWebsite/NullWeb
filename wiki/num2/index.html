NullMedia Bonus Features

How to post images

Step 1. Find an image online or upload a custom image to GitHub. If you are adding an image from online: Copy the following: <img src="linkhere"><br> Paste that into the body text field. Alternatively, press Ctrl + Shift + I to insert the image template.

Step 2. If on GitHub: Go to the image you uploaded on the spot where you can see the picture. Click "Raw" and copy the link to the website you get on. If you are adding an image from online: Find an image, right click it, and click Copy Image Address.

Step 3. Go back to the body text field and double click the part that says linkhere and paste the image address. Post it, and you're done!

How to make text bold

Before the text you want to be bold, type "<b>" (without the quotes), and put "</b>" after the bold text. Alternatively, press Ctrl + B to make text bold.

How to make text italics

Before the text you want to be italicized, type "<i>" (without the quotes), and put "</i>" after the italicized text. Alternatively, press Ctrl + I to make text italicized.

How to make text underlined

Before the text you want to be underlined, type "<u>" (without the quotes), and put "</u>" after the underlined text. Alternatively, press Ctrl + U to underline the text.

How to make text be in a code block

Before the text you want to be in a code block, type "<pre><code>" (without the quotes), and put "</pre></code>" after the code block. Alternatively, press Ctrl + Shift + C to insert a code block.

How to create a hyperlink

To insert a link manually, type "<a href="">" (without the quotes), then add the link and text you want inside the tag, and end with "</a>". Alternatively, press Ctrl + K to insert a link template like <a href="">[your title here]</a>, and the cursor will be placed inside the href attribute for easy linking.

How to create a blockquote

To insert a blockquote manually, type "<blockquote>" (without the quotes), and put "</blockquote>" after the blockquote. Alternatively, press Ctrl + Shift + Q to insert a blockquote.
