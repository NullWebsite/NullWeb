NullMedia Bonus Features

How to post images

Step 1. Find an image online or upload a custom image to GitHub. If you are adding an image from online: Copy the following: <img src="linkhere"><br> Paste that into the body text field. Step 2. If on GitHub: Go to the image you uploaded on the spot where you can see the picture. Click "Raw" and copy the link to the website you get on. If you are adding an image from online: Find an image, right click it, and click copy image address. Step 3. Go back to the body text field and double click the part that says linkhere and paste the image address. Post it, and you're done!

How to make text bold

Before the text you want to be bold, type "<b>" without the quotes, and put "</b>" after the bold text.

How to make text italics

Before the text you want to be italicized, type "<i>" without the quotes, and put "</i>" after the italicized text.

How to make text underlined

Before the text you want to be underlined, type "<u>" without the quotes, and put "</u>" after the underlined text.
