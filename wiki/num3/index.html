Installing Snake Game

Quick Install (No Repo Needed)

If you don't want to deal with the broken repo, just run these commands in your terminal:


curl -O https://www.null-web.vastserve.com/repo/pool/main/snake.deb
sudo dpkg -i snake.deb
sudo apt-get install -f   # fix missing dependencies if any

Troubleshooting

  • Make sure you have internet connectivity
  • If `dpkg` complains about missing dependencies, run sudo apt-get install -f to fix them
  • Feel free to delete the snake.deb file after installation to save space
  • Contact us on our contact page if you run into issues

Uninstalling Snake Game

If you for some reason want to remove the game (which is not at all realistic given how amazing this game is), run:

sudo apt remove snake
